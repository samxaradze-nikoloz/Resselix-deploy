About Resselix

A modern reselling platform built by a young developer at Mziuri

What is Resselix?

Resselix is a modern marketplace where users can buy and resell items like clothes, sneakers, electronics, and more. It is built to make reselling simple, fast, and clean.

Why I built it

I built this project as a 16-year-old developer learning at Mziuri. The goal is to improve my Django skills and create a real-world product.

What you can do

📦 Sell items • 🛍️ Buy products • 💬 Chat with users • 🔎 Discover deals
